# FORGE

Autonomous Backend Engineering Platform

## Overview

FORGE is a production-style autonomous backend engineering platform designed to understand repositories, plan engineering work, execute tasks, analyze failures, and continuously improve through deterministic workflows and AI-assisted reasoning.

## Goals

* Learn backend engineering deeply
* Learn system design
* Learn AI engineering
* Build production-grade workflows
* Showcase engineering skills

## Tech Stack

* Java 21
* Spring Boot
* PostgreSQL
* Docker
* GitHub Actions
* OpenTelemetry
* Ollama
